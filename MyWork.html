Memory Game

Memory Game is a simple game built using Python's Tkinter library. The project consists of six files:

  • Game.py - which handles overall construction and design of the in-game UI (primarily the grid and game tiles as well as the win/lose screen)
  • Game_Storage.txt - which handles data storage (I wrote the project before I learned sql or pandas so I was using file i/o for data storage and manipulation )
  • Home.py - which handles design of the home page, tutorial page and shop page
  • Run.py - which is the main program for the application
  • game_thread.py - which handles synchronization/multithreading between the game UI thread and timer thread
  • timer.py - which creates a simple countdown timer in the console
